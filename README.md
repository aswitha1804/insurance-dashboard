## 📊 Overview

**Insurance SafeGuard** is a professional, enterprise-grade insurance analytics portal that provides real-time insights into claims data, agent performance, fraud detection, and customer satisfaction metrics.

### ✨ Features

- 🔐 **Secure Login Portal** - Basic authentication with validation
- 📈 **Interactive Dashboard** - Real-time KPI metrics and charts
- 🌓 **Dark/Light Mode** - Toggle between themes with persistent storage
- 📱 **Fully Responsive** - Works on desktop, tablet, and mobile
- 🎨 **Modern Glassmorphism UI** - Beautiful gradient effects
- 📊 **Data Visualizations** - Chart.js integration for claims analytics
- 🚀 **IBM Cognos Integration** - Launch full analytics dashboard
- 📞 **Contact Page** - Interactive contact form

### 🎯 Key Metrics Tracked

| Metric | Value |
|--------|-------|
| Total Claims | 200+ |
